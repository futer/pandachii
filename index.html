About Pandachii Project

This aplication have been create for entertaiment a bunch for people. You will have option to feed Panda, change cloths, play some games or go to shopping. When the project will be finished there will be added more function.

This aplication will be improve with this steps :

1) Install and configure All environment, framework and other.
2) Set-up all main function to game
3) Made graphic for Pandachii
4) Add all main graphics and simple function to game.
5) Set-Up more function and improve other thing.
6) Set-Up and run test for this.
7) Update to more function and repair bug/error
8) Finish the game and check for other mistake
9) Create website for this project and add this to google shop.
